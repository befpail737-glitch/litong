## 🚀 部署说明

网站已准备好部署到 Cloudflare Pages。

### 部署步骤：

1. **推送到GitHub仓库**
   ```bash
   git init
   git add .
   git commit -m "Initial commit: LiTong Electronics website"
   git branch -M main
   git remote add origin https://github.com/befpail737-glitch/litong.git
   git push -u origin main
   ```

2. **Cloudflare Pages 部署配置**
   - 构建命令: `npm run build`
   - 输出目录: `out`
   - Node.js 版本: 18

3. **环境变量**（如需要）
   - 无需特殊环境变量

### 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 类型检查
npm run type-check

# 代码格式检查
npm run lint
```
