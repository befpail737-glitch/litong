# 功能测试指南

本文档列出了需要测试的功能和预期结果。

## 1. 管理后台表单修复测试

### 测试目标
验证管理后台的表单输入功能是否正常工作。

### 测试步骤
1. 访问 `/zh/admin/brands/`
2. 点击"新增品牌"按钮
3. 尝试在表单字段中输入文本
4. 验证所有输入框都能正常输入和编辑

### 预期结果
- ✅ 所有表单输入框都应该能够正常输入文本
- ✅ 下拉选择框能够正常选择选项
- ✅ 复选框能够正常切换状态

## 2. 文章管理品牌选择测试

### 测试目标
验证文章发布时能够选择关联品牌。

### 测试步骤
1. 访问 `/zh/admin/articles/`
2. 点击"新建文章"按钮
3. 查找"关联品牌"下拉选择框
4. 选择一个品牌
5. 填写其他必填字段并保存

### 预期结果
- ✅ 应该有"关联品牌"下拉选择框
- ✅ 下拉框中包含品牌选项（通用文章、STMicroelectronics、TI等）
- ✅ 能够成功选择品牌并保存文章

## 3. 前端文章展示测试

### 测试目标
验证发布的技术支持文章能在前端正确展示。

### 测试步骤
1. 首先在管理后台创建一篇关联到STMicroelectronics的技术文章
2. 访问品牌技术支持页面 `/zh/brands/stmicroelectronics/support`
3. 检查文章是否正确显示
4. 测试分类筛选功能

### 预期结果
- ✅ 技术支持页面显示文章卡片网格布局
- ✅ 文章显示正确的标题、摘要、作者、发布时间
- ✅ 分类筛选功能正常工作
- ✅ 推荐文章显示⭐标识

## 4. EXCEL产品上传测试

### 测试目标
验证改进的EXCEL上传功能能够自动处理任意列的数据。

### 测试步骤

#### 准备测试数据
创建一个Excel文件包含以下列：
- 型号/Part Number（必需）
- 品牌/Brand（可选）
- 描述/Description（可选）
- 工作电压/Supply Voltage（参数列）
- 封装/Package（参数列）
- 工作温度/Operating Temp（参数列）
- 价格/Price（可选）
- 库存/Stock（可选）

#### 测试流程
1. 访问 `/zh/admin/products/`
2. 点击上传Excel文件
3. 选择准备的测试文件
4. 查看上传结果
5. 检查产品列表中的新数据

### 预期结果
- ✅ 系统能够自动识别型号/Part Number列（即使列名不同）
- ✅ 所有非标准字段自动成为产品规格参数
- ✅ 上传成功后产品列表显示新数据
- ✅ 每个产品的规格参数正确显示

## 常见问题排查

### 如果表单无法输入
1. 检查页面是否配置了`export const dynamic = 'force-static'`
2. 确认组件是使用'use client'指令的客户端组件

### 如果文章不显示
1. 检查文章的status是否为'published'
2. 验证品牌slug匹配是否正确
3. 确认BrandSupport组件正确加载数据

### 如果EXCEL上传失败
1. 确认Excel文件包含"型号"、"名称"或"Part Number"列
2. 检查文件格式是否为.xlsx或.xls
3. 查看控制台错误信息
