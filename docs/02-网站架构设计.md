## 网站地图结构

### XML Sitemap规划
```xml
<!-- 静态页面 - 优先级1.0 -->
<url>
  <loc>https://www.elec-distributor.com/</loc>
  <changefreq>daily</changefreq>
  <priority>1.0</priority>
</url>

<!-- 品牌页面 - 优先级0.8 -->
<url>
  <loc>https://www.elec-distributor.com/brands/{brand-slug}</loc>

  <priority>0.8</priority>
</url>

<!-- 产品页面 - 优先级0.6 -->
<url>
  <loc>https://www.elec-distributor.com/products/{category-slug}</loc>
  <changefreq>monthly</changefreq>
  <priority>0.6</priority>
</url>

<!-- 内容页面 - 优先级0.5 -->
<url>
  <loc>https://www.elec-distributor.com/brands/{brand-slug}/support/{article-slug}</loc>
  <changefreq>monthly</changefreq>
  <priority>0.5</priority>
</url>
```

## 技术架构图

### Next.js App Router 结构
```
app/
├── layout.tsx              # 全局布局
├── page.tsx               # 首页
├── loading.tsx            # 加载组件
├── error.tsx              # 错误组件
├── not-found.tsx          # 404页面
├── brands/
│   ├── page.tsx           # 品牌列表页
│   └── [slug]/
│       ├── layout.tsx     # 品牌布局
│       ├── page.tsx       # 品牌首页
│       ├── about/
│       ├── categories/
│       ├── solutions/
│       ├── support/
│       └── news/
├── products/
│   ├── page.tsx           # 产品首页
│   └── [...slug]/
│       └── page.tsx       # 动态产品页
├── about/
│   ├── page.tsx           # 关于我们首页
│   └── [slug]/
│       └── page.tsx       # 关于我们子页面
└── [locale]/              # 多语言版本
    └── ...                # 复制上述结构
```

## 数据流架构

### 数据获取策略
```
Static Generation (SSG)    # 首页、品牌页、产品分类页
├── getStaticProps()      # 构建时数据获取
└── getStaticPaths()      # 动态路由生成

Server Side Rendering (SSR) # 搜索结果、用户相关页面
└── getServerSideProps()   # 请求时数据获取

Client Side Rendering (CSR) # 筛选、搜索交互
└── SWR/React Query       # 客户端数据管理
```

### Sanity数据同步
```
Sanity CMS
├── GROQ查询语言          # 数据查询
├── Webhooks触发          # 内容更新通知
├── CDN缓存策略           # 图片和文件加速
└── 增量静态再生成(ISR)    # 自动重新构建
```
