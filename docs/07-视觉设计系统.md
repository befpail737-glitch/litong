
## 响应式设计断点

### 断点系统定义
```scss
// 响应式断点
$breakpoints: (
  xs: 0,          // 超小设备 (手机竖屏)
  sm: 576px,      // 小设备 (手机横屏)
  md: 768px,      // 中等设备 (平板)
  lg: 992px,      // 大设备 (桌面)
  xl: 1200px,     // 超大设备 (大桌面)
  2xl: 1400px,    // 超超大设备
);

// 媒体查询 Mixins
@mixin mobile-only {
  @media (max-width: #{map-get($breakpoints, sm) - 1px}) {
    @content;
  }
}

@mixin tablet-up {
  @media (min-width: #{map-get($breakpoints, md)}) {
    @content;
  }
}

@mixin desktop-up {
  @media (min-width: #{map-get($breakpoints, lg)}) {
    @content;
  }
}

@mixin large-desktop-up {
  @media (min-width: #{map-get($breakpoints, xl)}) {
    @content;
  }
}
```

## 动效系统

### 过渡动画
```scss
// 基础过渡
$transitions: (
  fast: 0.15s ease-out,
  normal: 0.2s ease-in-out,
  slow: 0.3s ease-in-out,
);

// 动画效果类
.transition-fast {
  transition: all map-get($transitions, fast);
}

.transition-normal {
  transition: all map-get($transitions, normal);
}

.transition-slow {
  transition: all map-get($transitions, slow);
}

// 悬停效果
.hover-lift {
  transition: transform map-get($transitions, fast);
  
  &:hover {
    transform: translateY(-2px);
  }
}

.hover-scale {
  transition: transform map-get($transitions, fast);
  
  &:hover {
    transform: scale(1.05);
  }
}

// 渐入动画
@keyframes fadeIn {
  from {
    opacity: 0;
    transform: translateY(10px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}

.animate-fade-in {
  animation: fadeIn 0.3s ease-out;
}

// 加载动画
@keyframes spin {
  from {
    transform: rotate(0deg);
  }
  to {
    transform: rotate(360deg);
  }
}

.animate-spin {
  animation: spin 1s linear infinite;
}

// 脉冲动画（加载状态）
@keyframes pulse {
  0%, 100% {
    opacity: 1;
  }
  50% {
    opacity: 0.5;
  }
}

.animate-pulse {
  animation: pulse 2s cubic-bezier(0.4, 0, 0.6, 1) infinite;
}
```
