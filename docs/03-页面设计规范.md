# 页面设计规范

## 首页设计规范

### 页面结构
```
首页布局
├── Header (全局导航)
├── Hero Banner (英雄横幅)
├── Core Advantages (核心优势)
├── Product Areas (核心产品领域) 
├── Solutions (解决方案展示)
├── Latest News (最新动态)
├── Final CTA (最终行动号召)
└── Footer (全局页脚)
```

### Hero Banner 设计要求
**内容结构**:
```html
<section class="hero-banner">
  <div class="hero-content">
    <h1>电子元件代理专家 | 芯片现货供应商 - 力通电子</h1>
    <p class="hero-subtitle">专业电子元件代理商，提供正品原装芯片现货，技术支持与优势价格</p>
    <div class="hero-cta">
      <button class="primary-btn">立即咨询</button>
      <button class="secondary-btn">产品目录</button>
    </div>
  </div>
  <div class="hero-visual">
    <!-- 科技感背景图或产品展示 -->
  </div>
</section>
```

**SEO要求**:
- H1标签必须包含核心关键词"电子元件代理"
- 描述文案自然融入"芯片现货"、"代理商"等关键词
- 背景图片必须添加alt属性

1. **库存深度** - 现货库存，快速交付
2. **技术支持** - 专业FAE团队支持
3. **品质保证** - 数字证书追踪系统
4. **物流能力** - 全球物流配送网络

**设计要求**:
```html
<section class="advantages">
  <div class="advantage-card">

```

## 技术文章页设计规范

### 文章页面结构
```
技术文章页
├── 文章头部信息
├── 文章目录(TOC)
├── 文章正文内容
├── 相关产品链接
├── 相关文章推荐
├── 作者信息卡片
└── 分销商CTA
```

### 文章头部设计
```html
<header class="article-header">
  <div class="article-meta">
    <span class="article-category">{category}</span>
    <time datetime="{iso-date}">{publish-date}</time>
  </div>
  <h1 class="article-title">{title}</h1>
  <div class="article-excerpt">
    <p>{excerpt}</p>
  </div>
  <div class="article-author">
    <img src="/images/authors/{author-slug}.jpg" 
         alt="{author-name} 头像" />
    <div class="author-info">
      <span class="author-name">{author-name}</span>
      <span class="author-title">{job-title}</span>
    </div>
  </div>
  <div class="article-tags">
    {#each tags as tag}
      <a href="/tags/{tag-slug}" class="tag">#{tag-name}</a>
    {/each}
  </div>
</header>
```

### 文章内容区域
```html
<article class="article-content">
  <div class="article-toc">
    <h3>目录</h3>
    <ul>
      <li><a href="#section1">1. 引言</a></li>
      <li><a href="#section2">2. 技术原理</a></li>
      <li><a href="#section3">3. 应用案例</a></li>
    </ul>
  </div>
  
  <div class="article-body">
    <!-- 富文本内容 -->
    <h2 id="section1">1. 引言</h2>
    <p>文章内容...</p>
    
    <!-- 产品链接示例 -->
    <p>推荐使用 
      <a href="/products/ic/microcontroller/stm32f103c8t6" 
         class="product-link">STM32F103C8T6</a> 
      微控制器实现此功能。
    </p>
  </div>
</article>
```
