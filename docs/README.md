## 🔧 开发指南

### 环境要求
```bash
Node.js >= 18.0.0
npm >= 9.0.0
Git >= 2.30.0
```

### 快速开始
```bash
# 1. 克隆仓库
git clone https://github.com/befpail737-glitch/litong.git

# 2. 安装依赖
cd litong
npm install

# 3. 配置环境变量
cp .env.example .env.local
# 编辑 .env.local 填入必要配置

# 4. 启动开发服务器
npm run dev

# 5. 访问应用
# 前端: http://localhost:3000
# CMS后台: http://localhost:3000/admin
```

### 开发规范
- 遵循TypeScript严格模式
- 使用ESLint和Prettier保持代码风格
- 编写单元测试覆盖关键功能
- 提交前运行类型检查和测试
- 使用语义化的Git提交信息
